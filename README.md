# DriveMarket (Car Classifieds)

Welcome to **DriveMarket**, a premium car classifieds marketplace.

> **Note:** This is a **hobby project** built for personal portfolio and educational purposes to explore modern web technologies and UI/UX design.

## Features

- **Modern UI:** Built with [Next.js](https://nextjs.org), utilizing a high-trust, professional aesthetic with neutral color palettes and glassmorphism elements.
- **Smooth Animations:** Powered by [Framer Motion](https://www.framer.com/motion/) for fluid page transitions and interactive hover effects.
- **Responsive Design:** Fully mobile-responsive layout crafted with [Tailwind CSS](https://tailwindcss.com), ensuring a seamless experience across all devices.
- **Analytics:** Integrated with [Vercel Analytics](https://vercel.com/analytics) to monitor page views and core web vitals.
- **Component Library:** Built with customized UI components, utilizing class-variance-authority and clsx for scalable styling.

## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS, PostCSS
- **Animations:** Framer Motion, tw-animate-css
- **Icons:** Lucide React
- **Deployment:** Vercel
